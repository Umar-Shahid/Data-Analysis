## üìä Featured Projects

### 1. üö¥ Cyclistic Bike Share Analysis
**[View Notebook](https://www.kaggle.com/muhammadumarshahid) | [View Dashboard](https://public.tableau.com/app/profile/muhammad.umar.shahid)**

Comprehensive analysis of 500,000+ bike-share trip records to identify customer behavior patterns and inform marketing strategy.

**Business Problem:**  
Cyclistic wants to convert casual riders into annual members to increase profitability and reduce customer acquisition costs.

**My Approach:**
- Cleaned and validated 500,000+ trip records using Python (Pandas, NumPy)
- Performed exploratory data analysis examining ride duration, frequency, timing patterns
- Used SQL to aggregate data by user type, day of week, and time periods
- Created interactive Tableau dashboard for stakeholder presentation
- Conducted statistical analysis to quantify segment differences

**Key Findings:**
- Annual members show **35% higher weekday usage**, indicating commuter behavior
- Casual riders take **2x longer trips** on average (28 min vs 14 min)
- Weekend usage patterns differ dramatically: **60% spike for casuals vs 15% for members**
- Peak hours reveal distinct use cases: Members peak at 8am/5pm (commute), casuals at 2-4pm (leisure)

**Business Impact:**
Delivered three data-driven recommendations:
1. Target weekend casual riders with conversion campaigns emphasizing commute savings
2. Offer trial memberships during peak casual usage months (June-August)
3. Create weekend-only membership tier to capture leisure riders

**Projected Result:** 15-20% increase in membership conversions

**Tech Stack:** Python (Pandas, NumPy, Matplotlib), SQL, Tableau

### 3. üçΩÔ∏è Restaurant Tips Analysis
**[View on Kaggle](https://www.kaggle.com/muhammadumarshahid)**

Statistical analysis examining factors influencing tipping behavior in restaurants using regression modeling.

**Research Questions:**
- What factors most strongly predict tip amount?
- Is there a relationship between bill size and tip percentage?
- How does party size affect tipping behavior?
- Do temporal factors (day/time) influence tips?

**Methodology:**
- Regression analysis to identify tip predictors
- Hypothesis testing for statistical significance (p-values, confidence intervals)
- Correlation analysis between variables
- Data visualization to communicate patterns

**Key Findings:**
- **Strong positive correlation** between bill amount and tip (r=0.68, p<0.001)
- Party size **negatively correlated** with tip percentage
- Dinner service receives **12% higher tips** than lunch on average
- No statistically significant difference in tipping between weekdays/weekends

**Statistical Methods:**
- Linear regression modeling
- Pearson correlation analysis
- T-tests and ANOVA
- Residual analysis for model validation

**Tech Stack:** Python (Pandas, SciPy, Seaborn), Statistical Hypothesis Testing, Regression Analysis

### 4. üì± Waze User Behavior & Churn Analysis
**[View on Kaggle](https://www.kaggle.com/muhammadumarshahid)**

User retention analysis to identify churn indicators and support product strategy decisions for navigation app.

**Business Context:**  
Understanding why users stop using Waze to improve retention rates and user lifetime value.

**Analysis:**
- Exploratory data analysis of user engagement patterns
- Churn prediction using logistic regression
- Feature importance identification
- Cohort analysis to track retention over time

**Key Findings:**

- Navigation accuracy issues identified as primary churn driver
- Active notification engagement **reduces churn by 40%**
- First-week engagement is strong predictor of long-term retention

**Recommendations:**
1. Implement early warning system for low-engagement users
2. Improve navigation accuracy in high-churn regions
3. Optimize notification strategy to increase engagement without annoying users
4. Create onboarding flow to drive first-week usage

**Tech Stack:** Python (Pandas, NumPy, Scikit-learn), Statistical Analysis, Predictive Modeling
